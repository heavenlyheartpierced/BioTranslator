program ideas

what is being tested:
1. ability to read # of codons in a strain (1, 3, 5, 7, 10) (including start-stop codons)
2. ability to recognize coding and non-coding sections (start is always AUG)

idea iteration one
1: reads one codon
2: input a number of codons to read in a string
3: recognizes the introns and exons in a string

1-codon strain: AUG
3-codon strain: AUG CUG UGA
5-codon strain: AUG UCU UGG CAA UAG
7-codon strain: AUG AAU GAC CAC GCC AUA UAA
10-codon strain: AUG AGU UAU CAG CGG AUC GUG UGC UGG UGA

1-exon strain: GUG AAA CUA (AUG CUC UGA) UUU CGG GAC
2-exon strain: CCC GGA UCU (AUG GUG UAA) GAC AAG AUA (AUG GCC UAG) GCA AGC AAU
