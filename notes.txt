what is being tested:
1. ability to read # of codons in a strain (1, 3, 5, 7, 10)
2. ability to recognize coding and non-coding sections (start is always AUG)

1-codon strain: AUG (expected to return Methionine)
3-codon strain: AUGCUGUGA (expected to return Methionine-Leucine, stops)
5-codon strain: AUGUCUUGGCAAUAG (expected to return Methionine-Serine-Tryptophan-Glutamine, stops)
7-codon strain: AUGAAUGACCACGCCAUAUAA (expected to return Methionine-Asparagine-Aspartic acid-Histidine-Alanine-Isoleucine, stops)
10-codon strain: AUGAGUUAUCAGCGGAUCGUGUGCUGGUGA (expected to return Methionine-Serine-Tyrosine-Glutamine-Arginine-Isoleucine-Valine-Cysteine-Tryptophan, stops)

1-exon strain: GUGAAACUAAUGCUCUGAUUUCGGGAC (expected to return Methionine-Leucine, stops)
2-exon strain: CCCGGAUCUAUGGUGUAAGACAAGAUAAUGGCCUAGGCAAGCAAU (expected to return Methionine-Valine-Methionine-Alanine, stops)
